Development of web application for calculation of temperature and composition
in Liquid Phase Epitaxy technique for Mercury-Cadmium-Telluride IR Detectors

You can check out the website here . Play around with various input values and observe how the ouput changes.
The Hg and Cd inputs are the mole fractions of Mercury and Cadmium respectively. T is temperature and X is Cadmium composition.
These values are important for carrying out meaningful work in Liquid Phase Epitaxy (LPE) Labs.

Project Overview

I created a dataset of about 100 values of Hg, Cd, T and X values. This dataset was fed into a Multivariate Polynomial Regression algorithm. Using this algorithm, a six degree best fit function for the input dataset was created which approximates the values of T and X within 5-10% error. This six degree function works in the backend and its output is what you see as T and X in the main web site.

For the backend, I used Python and a popular web microframework Flask. The six degree polynomial equation (which contains a whopping 28 terms!) was coded in the backend. It takes values of Hg and Cd mole fractions as input and computes T and X according to the formula. The website and all its python code is hosted at PythonAnywhere.

The frontend is made using the Big 3 of the web - HTML, CSS and JavaScript. Along with this, the framework Bootstrap was also used to design the website. The frontend is made deliberately dark to give it a more robust and modern look and feel to it.
